# Blackjack Game 🎲♠️

A simple Blackjack game built using HTML, CSS, and JavaScript. Players can start a game, draw new cards, and try to reach a total of **21** without going over.

## 🚀 Features

- Start a new Blackjack game
- Draw additional cards
- Automatic score calculation
- Blackjack detection (21 points)
- Bust detection (score greater than 21)
- Clean and responsive user interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 🎮 How to Play

1. Click the **START GAME** button to begin.
2. Two random cards are dealt to the player.
3. The total card value is calculated automatically.
4. Click **NEW CARD** to draw another card.
5. Try to reach exactly **21** without exceeding it.

### Game Rules

- If your total is less than 21, you can draw another card.
- If your total becomes exactly 21, you win with **Blackjack**.
- If your total exceeds 21, you lose (Bust).
