Blackjack Game 🎲♠️

A simple Blackjack game built using HTML, CSS, and JavaScript. Players can start a game, draw new cards, and try to reach a total of 21 without going over.

🚀 Features
Start a new Blackjack game
Draw additional cards
Automatic score calculation
Blackjack detection (21 points)
Bust detection (score greater than 21)
Clean and responsive UI

CSS
JavaScript

🎮 How to Play
Click START GAME to receive two random cards.
The sum of the card values is calculated automatically.
If your total is:
Less than 21 → You can draw a new card.
Exactly 21 → You get Blackjack!
Greater than 21 → You Bust and lose the game.
Click NEW CARD to draw another card while still in the game.
